# Watershed Sentinel

DGL233 Project 3

## Project Overview

This project was about recreating a custom WordPress theme inspired by the Watershed Sentinel environmental magazine website. The site was built using custom PHP templates, WordPress template hierarchy, custom post types, dynamic loops, reusable template parts, and responsive CSS.

The project includes:

* Custom homepage with dynamic content loops
* Blog archive page
* Single blog post templates
* Custom Issues post type
* Issues archive page
* Single issue templates
* Contact page template
* 404 page template
* Responsive mobile layouts
* Custom navigation and footer menus

## Development Challenges

One of the biggest challenges during development was understanding how WordPress template hierarchy works and making sure that the correct template files loaded for archives and custom post types.

Another challenge was creating reusable card layouts that worked consistently across blog posts and Issues content while maintaining clean spacing and responsive behavior.

Debugging CSS spacing, card alignment, navigation styling, and permalink/template loading issues also required careful testing and refinement.
